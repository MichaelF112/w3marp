# w3marp
## A web3 market place built on the stacks blockchain

### Tools
This product was built with `Clarity` and `React`

### Functionalities
| Functionality | Detail |
|---------- | ------------------- |
| List Item | The platform gives users the ability to list products for sale |
| Buy Item | Users search and purchase an item of their choice |
| Invoicing | Each product is bought and sold using invoices (only payment that matched the invoice price will be sold) |
